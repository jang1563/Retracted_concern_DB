# Code Of Conduct

## Our Commitment

We are building a triage-benchmark harness that touches named researchers' reputations. That puts a higher bar on how we talk to each other and how we talk about third parties in issues, PRs, and discussions.

## Expected Behavior

- Treat contributors, reviewers, and subjects of integrity records with professional respect.
- Keep discussion focused on evidence, methodology, governance, and code.
- Assume good faith; ask clarifying questions before attributing motive.
- Disclose relevant conflicts (employment, affiliation, authorship on discussed records).

## Unacceptable Behavior

- Public allegations or judgments of misconduct against named researchers, institutions, journals, or publishers. Such content will be removed.
- Harassment, personal attacks, or discriminatory language.
- Posting private communications or non-public documents.
- Using issues or PRs to amplify unresolved integrity concerns instead of the documented dispute workflow.
- Publishing model outputs, rankings, or scores derived from this project in a way that violates [ETHICS.md](ETHICS.md).

## Enforcement

Maintainers may remove comments, close issues, or restrict repository access when behavior is inconsistent with this document. Repeated or severe violations will result in a permanent ban.

## Reporting

- Report conduct issues privately to the maintainer listed in [CITATION.cff](CITATION.cff), or via a GitHub report on the offending content.
- Record-level integrity disputes go through the public site's dispute contact, not through this channel.
- Security issues go through [SECURITY.md](SECURITY.md).

## Attribution

This Code of Conduct is adapted from the spirit of the Contributor Covenant, with additional clauses specific to research-integrity tooling.
